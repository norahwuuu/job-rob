===== Job Description =====
The Role

We are looking for a hands-on Growth Engineer to design, implement, and manage scalable API integrations that connect HelloFresh with external partner ecosystems including telecoms, financial providers, loyalty programs, and health/wellness platforms.

This role combines technical execution (building and deploying APIs, data workflows, and automation) with project ownership (defining requirements, managing partners, ensuring delivery).

You will own integrations from design to production, delivering end-to-end partner connectivity that enables subscription bundling, loyalty rewards, account linking, and nutrition data exchange, all while maintaining reliability, compliance, and a seamless customer experience.

What You'll Do
Integration Implementation: Build and maintain RESTful APIs and webhooks to connect HelloFresh with partner platforms for loyalty rewards and account linking. 
Execution & Support: Develop automation scripts and data workflows to handle recurring syncs and reporting for partner programs. 
Technical Documentation: Maintain high-quality API documentation and partner developer guides to ensure smooth onboarding. 
Reliability: Troubleshoot issues, manage error handling, and ensure the performance of existing integrations. 
Collaboration: Work cross-functionally with Product, Data, and Finance teams to implement business logic for revenue-sharing or benefit application. 
What You’ll Bring

Technical Skills: * Solid experience with RESTful API development (Python/Node.js—familiarity with FastAPI is a plus).
Good understanding of OAuth2, webhooks, and how to manage secure data exchange. 
Experience using API testing tools like Postman and documenting work via Swagger. 
Familiarity with AWS (containers) and working knowledge of SQL/NoSQL databases. 
Preferred Experience: * Interest or previous exposure to B2B SaaS, fintech, or subscription-based platforms.
Familiarity with tools like Snowflake, Contentful, or N8N is a bonus. 
Soft Skills:
Ownership Mindset: A proactive approach to completing tasks and ensuring code quality. 
Organizational Skills: The ability to manage multiple integration tasks and meet deadlines in a fast-paced environment.
